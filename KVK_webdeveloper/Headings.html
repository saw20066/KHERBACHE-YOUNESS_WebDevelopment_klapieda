This is Heading Level 1

Think main page title. Bold, huge, owns the top spot.

This is Heading Level 2

Think section title. Still important, but a notch below H1.

Major sections, like "Introduction" or "How-To Guide."

This is Heading Level 3

Think subsection title. A bit smaller, but still stands out.

Used for subsections within H2 sections, like "Step 1" or "Details."

This is Heading Level 4

Think sub-subsection title. Smaller, but still noticeable.

For further breakdowns within H3 sections, like "Part A" or "Example 1."

This is Heading Level 5

Think minor heading. Small, but still a heading.

Rarely used, but can help organize complex content.

This is Heading Level 6

Think tiny heading. The smallest, least prominent heading.

Very rarely used, but can be useful for very detailed documents.
